# Knowledge Vault

Персональная база знаний, управляемая через Obsidian + Claude Code.

## Структура папок

- **Resources/** — исследования, статьи, конспекты, референсы
- **Tasks/** — текущие задачи и чеклисты
- **Ideas/** — идеи, гипотезы, черновики
- **Projects/** — активные проекты с архитектурой и решениями
- **Daily/** — ежедневные заметки
- **Templates/** — шаблоны для быстрого создания заметок
- **Archive/** — завершенные или неактуальные заметки

## Соглашения

- Все заметки содержат YAML frontmatter с `type`, `tags`, `status`
- Связи между заметками через `[[wiki-links]]`
- Теги: `#research`, `#idea`, `#project`, `#task`, `#reference`
